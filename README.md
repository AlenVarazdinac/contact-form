# Contact form

## Runtime requirements

```
NodeJS - version 8.9 or above (v10+ recommended)
```

## Project setup

1. Clone project

```

git clone git@github.com:AlenVarazdinac/contact-form.git ContactFormTask

```

2. Install packages

```

npm install

```

3. Compile and serve the project

```

npm run serve

```

### Run unit tests

```

npm run test:unit
```
